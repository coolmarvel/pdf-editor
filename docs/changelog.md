## v1.2.4 — 2026-07-09

- 지우개(도형): 드래그하는 즉시 **흰 사각형이 실시간으로 늘어나며 덮임** (기존: 파란 선으로 영역만 표시되다가 놓으면 덮임)
- 그리기 지우개: 형광펜 등이 뭉텅이로 지워지던 것 → 브러시가 닿은 만큼만 매끄럽게 지워짐

## v1.2.3 — 2026-07-09

- 설치 화면의 "파일 변환기(File Converter)" 문구를 "PDF 편집기"로 수정

## v1.2.2 — 2026-07-08

- **그리기 지우개**: 선 전체가 아니라 **브러시가 닿은 부분만** 지워짐 (선이 토막으로 나뉨)
- **지우개 ↔ 도형 버튼 연동**: 지우개를 켜면 도형 버튼이 "사각형"으로 리셋되며 함께 활성화,
  그 상태에서 도형 드롭다운으로 지우개 모양(사각형/원) 변경. 그리기 지우개 선택 시 도형 버튼 해제
- 도구를 바꿀 때 문서가 몇 픽셀 위아래로 튀던 문제 수정 (컨텍스트 툴바 높이 고정)

## v1.2.1 — 2026-07-08

- **지우개**: 드래그로 사각형/원을 그려 덮어서 지움 — 테두리 색·두께·선 스타일(실선/점선/파선)·채우기·불투명도
  변경 가능, 그린 뒤 바로 선택돼 옵션 바에서 스타일링, 도구는 유지(연속 지우기)
- **그리기 지우개**: 페이지에 낙서(연필·형광펜)가 있을 때만 속 빈 원 커서로 바뀌고, 문지르면 지워짐
- 사각형/원 도형에도 테두리 선 스타일·기본 채우기 옵션 추가

## v1.1.2 — 2026-07-08

텍스트 수정을 Guru 처럼 "세션 모드"로.

- 텍스트 수정 버튼이 켜져 있는 동안 하이라이트 유지 — 여러 텍스트를 연달아 수정/이동/삭제
- 삭제 시 "이 텍스트 상자를 지울까요?" 확인 → 지우면 그 글자가 문서에서 지워진 상태가 됨
- 모드 중에는 실행취소/다시실행 비활성 (변경은 버퍼에 보관)
- 모드를 끄면 변경이 있을 때만 "수정한 내용을 저장할까요?" (계속 편집/저장 안 함/저장) — 저장 안 함 = 전부 원복,
  저장 후에는 Ctrl+Z 한 번으로 세션 전체 되돌리기 가능

## v1.1.1 — 2026-07-08

## v1.1.0 — 2026-07-08

**텍스트 추가 기능 확정** (사용자 선언) + 텍스트 수정 Guru 흐름 1차.

- 텍스트 수정: 글자 위 호버 시 포인터 커서 → 클릭한 채 드래그 = 이동(이동 커서), 놓으면 클릭 지점에 캐럿 → 바로 수정
- 옮겨도 원본 글자는 흰 커버가 계속 가림 (커버와 텍스트 상자 분리)
- 편집 상자가 "팝업"이 아니라 글자 크기 그대로 제자리에서 열림 (그림자 제거·투명 배경)
- 텍스트 수정 객체에도 8점 핸들(크기 조절)
- 커서 규칙 통일: 객체 호버 = 포인터, 잡음/드래그 = 이동 — 텍스트 추가 객체 포함 전체 적용

- 색 팔레트(프리셋+직접 선택)·글자 배경 팔레트(+없음)·불투명도 %/슬라이더·폰트 17종(미리보기)
- **세로 정렬**(위/가운데/아래) — 상하 핸들로 상자를 늘린 뒤 박스 안에서 정렬. 가로 정렬과 조합 가능

- Playwright 시각 검증 도입(e2e/visual-check.cjs) — 릴리스 전 자동 스크린샷·픽셀 계측

## v1.0.2 — 2026-07-08

텍스트 도구 UX 재작업 (2차 피드백).

- **텍스트 추가 원샷**: 한 번 찍으면 도구가 자동 해제 — 박스가 연달아 생기지 않음
- **입력 박스 재디자인**: 커서 크기로 시작해 타이핑한 만큼 자라는 깔끔한 연파랑 박스 (Guru 스타일)
- **선택 UI 재작업**: 파란 점 6개 + 회전 점 1개 — 모서리 점 = 글자 크기 조절, 좌우 점 = 폭, 하단 점 = 360° 회전
- **회전·리사이즈 실동작 수정**: 핸들이 클릭을 못 받던 레이어 순서 버그 해결 (이미지/도형 리사이즈도 이번에 처음 실제로 동작)

## v1.0.1 — 2026-07-08

첫 설치 테스트 피드백 반영 (스크린샷 3장).

- **텍스트 추가 수정**: 클릭해도 입력 박스가 안 뜨던 버그 수정 (생성 직후 포커스를 뺏겨 즉시 삭제되던 문제)
- **텍스트 상자 강화**: 글자 배경색, 하단 손잡이로 360° 자유 회전(45° 근처 스냅), 선택 상태에서 정렬 버튼 추가
- **하단 페이저 정리**: "페이지:"가 세로로 깨지던 것 가로 고정, 실수로 100% 폭으로 그려지던 회색 박스 제거, "1/32" 글씨 확대, 반픽셀 블러 제거
- **글씨 선명도**: 숫자·영문을 Segoe UI로 렌더(작은 크기에서 Malgun Gothic 깨짐 완화), 툴바 라벨 10→11px

첫 릴리스. pdfguru.com 벤치마킹 PDF 편집기.

- PDF 열기(선택/드래그) → 로딩 화면 → 편집기
- 편집 도구: 텍스트 추가/**기존 텍스트 수정**, 연필·형광펜, 지우개(흰칠)·그리기 지우개, 사각형·원·X·체크, 이미지, 스탬프(프리셋+커스텀), 서명(그리기/이미지/타이핑), 스티키 노트, 링크(웹/페이지)
- 페이지 관리: 새 페이지·삭제·복제·회전·순서 이동·다른 PDF 가져오기
- 실행취소/다시실행(Ctrl+Z/Y), 삭제(Delete), 줌·손 도구, 첫 화면 맞춤 배율
- 저장(원본 벡터 유지, 주석은 실제 PDF 주석), 인쇄
